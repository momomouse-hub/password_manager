# 提出QUESTパスワードマネージャー

## ステップ1

パスワードを保存する処理を、シェルスクリプトで作成。

- サービス名
- ユーザー名
- パスワード

が保存できるように。

シェルスクリプトを実行すると、以下が表示される。

```bash
パスワードマネージャーへようこそ！
サービス名を入力してください：
ユーザー名を入力してください：
パスワードを入力してください：
# 入力が完了したら
Thank you!
```

サービス名、ユーザー名、パスワードの入力が求められ、入力された情報を"サービス名:ユーザー名:パスワードという形式でファイルに保存。シェルスクリプトを実行するたびに入力された情報はファイルに追記されるように。
